Kontak Kami

Ada pertanyaan atau butuh bantuan? Tim admin MIKU Store siap membantu Anda.

Cara Menghubungi Admin

Silakan hubungi admin MIKU Store melalui kontak resmi yang tersedia di halaman utama atau yang tercantum pada invoice transaksi Anda.

Untuk mempercepat proses penanganan, siapkan nomor invoice transaksi Anda sebelum menghubungi admin.

Jam Layanan

Tim admin MIKU Store berusaha merespons setiap pertanyaan secepat mungkin. Estimasi respons:

  • Pertanyaan umum: dalam 1×24 jam
  • Klaim refund / kendala transaksi: dalam 1×24 jam kerja
  • Status Perlu Review Admin: 1×24 jam atau tergantung respons provider

Informasi yang Perlu Disiapkan

Agar admin dapat membantu lebih cepat, siapkan informasi berikut saat menghubungi:

  • Nomor Invoice transaksi (contoh: TRX-1234567890)
  • Status transaksi yang terlihat di dashboard atau halaman Cek Transaksi
  • ID Game / ID Server yang digunakan saat pemesanan
  • Screenshot atau bukti transaksi jika ada

Cek Status Transaksi Sendiri

Sebelum menghubungi admin, Anda bisa cek status transaksi secara mandiri melalui:

  • Halaman Cek Transaksi — masukkan nomor invoice untuk melihat status terkini.
  • Dashboard — login untuk melihat riwayat lengkap transaksi Anda.
