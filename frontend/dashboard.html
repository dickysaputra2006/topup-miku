Cara Deposit Saldo

Transfer ke salah satu metode di bawah, lalu ajukan request dengan nominal yang sesuai. Saldo masuk setelah admin verifikasi.

GoPay Nomor akan diinformasikan oleh admin MIKU Store.
DANA Nomor akan diinformasikan oleh admin MIKU Store.
ShopeePay Nomor akan diinformasikan oleh admin MIKU Store.
SeaBank Nomor akan diinformasikan oleh admin MIKU Store.

Penting: Saldo tidak bertambah otomatis. Admin akan memverifikasi pembayaran dan menyetujui deposit.

Ajukan Request Deposit

Minimum deposit Rp 10.000. Nominal akhir akan ditambah kode unik 3 digit untuk memudahkan verifikasi.

Instruksi Pembayaran

Setelah transfer, tunggu konfirmasi dari admin. Riwayat request terlihat di bawah.
